## Sommersprint – Das Bildungs-Reallabor für agile Sanierung

## Projektübersicht

„Sommersprint“ ist ein innovatives Prozessmodell zur radikalen Beschleunigung kommunaler Sanierungsvorhaben. Das Projekt kombiniert moderne IT-Logik (BIM/LiDAR) mit agiler Nachwuchsförderung im Handwerk. Digitale Architektur (Die App): Ein Smart-Sanierungs-Assistent, der die Lücke zwischen Baustelle und Planung schließt.
Features: LiDAR-Scan, automatisches Fitting, Multi-Cart-System und Ressourcenplanung via JSON/IFC-Export.

Ursprünglich als Pilotprojekt für die Almenhof-Grundschule in Mannheim konzipiert und durch die lokale Wirtschaftsförderung fachlich validiert, fungiert das Konzept heute als überregionale Blaupause (Framework) für effiziente Gebäudemodernisierung.  

Skalierbarkeit: Das Framework ist mandantenfähig und kann auf verschiedene Standorte und Kommunen übertragen werden.
